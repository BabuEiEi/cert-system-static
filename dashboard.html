แผงควบคุมเกียรติบัตร

คู่มือระบบเกียรติบัตรออนไลน์

สำหรับผู้ดูแลระบบและเจ้าหน้าที่

เริ่มต้นใช้งาน

  1. 1. เลือกกิจกรรม
    เลือกจากรายการด้านซ้าย หากยังไม่มี กิจกรรมใหม่ต้องสร้างโดย Admin
  2. 2. ออกแบบเกียรติบัตร
    ตั้งค่าพื้นหลัง โลโก้ ข้อความ ฟอนต์ สี และลายเซ็น แล้วกดพรีวิว
  3. 3. บันทึกรูปแบบ
    กด “บันทึกรูปแบบ” ก่อนเผยแพร่ เพื่อให้หน้าเว็บสาธารณะใช้ค่าล่าสุด
  4. 4. นำเข้ารายชื่อ
    วางชื่อหนึ่งคนต่อหนึ่งบรรทัด ตรวจปีและเลขที่ถัดไป แล้วจึงนำเข้า
  5. 5. ตรวจสอบไฟล์
    ทดลองพรีวิวและสร้าง PNG อย่างน้อยหนึ่งรายก่อนสร้าง ZIP ทั้งหมด
  6. 6. เผยแพร่
    Admin เปิด “เผยแพร่ให้ค้นหาได้” เมื่อรูปแบบและรายชื่อถูกต้องแล้ว

เลขที่เกียรติบัตร

  • เว้นช่องเลขเริ่มต้นว่าง ระบบจะต่อจากเลขล่าสุดอัตโนมัติ
  • ระบบเก็บตัวนับแยกตามปี พ.ศ. และใช้ transaction ป้องกันเลขซ้ำ
  • ห้ามกรอกเลขเริ่มต้นต่ำกว่าเลขถัดไป ระบบจะไม่อนุญาต
  • การลบรายชื่อไม่ทำให้เลขล่าสุดย้อนกลับ เพื่อไม่ให้นำเลขเดิมมาใช้ซ้ำ
  • ตรวจคำนำหน้า ปี พ.ศ. และตัวเลือกเลขไทยก่อนนำเข้าเสมอ

หน้าเว็บสำหรับผู้รับ

  • แสดงเฉพาะกิจกรรมที่ Admin เผยแพร่แล้ว
  • ค้นชื่อหรือนามสกุลเพียงบางส่วนได้
  • ดูตัวอย่าง ดาวน์โหลด PNG และแจ้งแก้ไขผ่าน Google Form ได้
  • หากแก้รูปแบบแล้ว ต้องกดบันทึกก่อน หน้าเว็บสาธารณะจึงเห็นค่าล่าสุด

ข้อจำกัดและข้อควรระวัง

  • นำเข้ารายชื่อได้สูงสุด 450 คนต่อครั้ง แต่หนึ่งกิจกรรมนำเข้าได้หลายรอบ
  • กิจกรรมที่มีหลายพันรายชื่ออาจโหลดช้าบนอุปกรณ์พกพา เพราะ Dashboard และหน้าค้นหาดึงรายชื่อมาประมวลผลในเบราว์เซอร์
  • การสร้าง ZIP จำนวนมากใช้หน่วยความจำสูง แนะนำแบ่งสร้างเป็นช่วงเมื่อมีรายชื่อจำนวนมาก
  • รูปพื้นหลัง โลโก้ และลายเซ็นต้องเป็น URL ที่เปิดอ่านจากภายนอกได้และรองรับ CORS มิฉะนั้น PNG อาจสร้างไม่สำเร็จ
  • การลบกิจกรรมจะลบรายชื่อและรูปแบบทั้งหมดถาวร ควรตรวจสอบและสำรองข้อมูลก่อน
  • รายชื่อหรือเลขที่ซ้ำซึ่งเกิดก่อนระบบตัวนับอัตโนมัติจะไม่ถูกแก้ย้อนหลัง ต้องตรวจและแก้ข้อมูลเดิมแยกต่างหาก
